# 学习推进工具

本目录保存执行 `AI算力能源财税与碳合规` 一次学习的互动提示词和 harness。它只约束学习纪律，不定义研究主线；研究路线以 PPT 原始材料及其对象网络为准。

## 文件

- [[一次学习推进提示词]]：给 Agent 的运行边界和推进顺序。
- [[一次学习推进-harness]]：每轮开始、推进、收束和写入前的检查表。

## 使用边界

- 只服务 `01-一次学习/`，不进入 `02-二次学习/`。
- 不替代 PPT 图片、OCR、页索引、研究对象地图、概念库、主转换链和反馈回路。
- AI 服务规格、工作流和任务集合只作为局部接口工具，不作为默认主线。
- 不把一次学习变成逐题答题、概念定义考试或证据审计。
- 只有用户完成自己的复述后，才写入一次学习笔记。
