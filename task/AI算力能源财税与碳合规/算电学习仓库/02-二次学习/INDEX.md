# 二次学习索引

二次学习用于纠正和重建一次学习，不是把一次笔记重新排版。它至少要引入一种新证据或新视角，并把自己的旧解释摆在旁边比较。

## 当前入口

- [[2026-07-15-第一轮主链二次学习]]

## 进入条件

- 已完成对应的一次学习和关闭资料复述；
- 已明确至少一个待核验连接；
- 已选择能帮助核验的来源，而不是继续泛读。

## 结束条件

- 明确写出“第一次哪里错了或过于简单”；
- 能用自己的结构重建关系，不依赖原材料顺序；
- 将稳定定义写回[[概念地图|概念区]]，将跨概念解释写入[[05-逻辑链条/INDEX|逻辑链条]]；
- 仍不能回答的问题进入[[06-学习问题/INDEX|问题索引]]。
