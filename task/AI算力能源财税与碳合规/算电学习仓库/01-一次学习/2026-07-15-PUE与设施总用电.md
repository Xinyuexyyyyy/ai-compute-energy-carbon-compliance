# PUE 与设施总用电

## 本轮主体关系与核心问题

关系：IT 用电与设施附加用电 -> 数据中心总用电 -> PUE。

核心问题：IT 用电不变而冷却用电增加时，PUE 怎样变化。

## 用户的初始理解

冷却系统用电不属于 IT 用电；总用电增加、IT 用电不变时，PUE 会变高，设施效率变差。

## 追问中暴露的混淆

- PUE 的公式是设施总用电除以 IT 用电，不是 IT 用电除以设施总用电。
- 总用电增加不等于数据中心总容量增加；前者是运行结果，后者是设计或可用能力。
- PUE 反映设施附加能耗比例，不能单独判断任务算效、任务碳排或 GPU 是否高效。

## 共同形成的解释

```text
PUE = 数据中心设施总用电 ÷ IT 设备用电

设施总用电
= IT 用电
+ 冷却用电
+ 供配电损耗
+ 其他设施用电
```

在 IT 用电保持不变时，冷却或其他设施用电增加会提高分子，使 PUE 数值升高，表示设施附加能耗相对增加、设施效率变差。PUE 理论下限接近 1，但实际计算必须统一时间范围、测点和设施边界。

## 最小算例

```text
方案 A：IT 用电 100 kWh，设施总用电 130 kWh，PUE = 1.30
方案 B：IT 用电 100 kWh，设施总用电 150 kWh，PUE = 1.50
```

方案 B 的 IT 工作量未变，但冷却或其他设施附加用电更多，因此 PUE 更差。

两个数据中心 PUE 相同，只说明“设施总用电与 IT 用电的比例”相同，不能说明同一任务的单位任务电量或碳排相同。任务、模型、输入规模、GPU 和运行效率会先决定单位任务 IT 电量；PUE 再决定设施附加能耗；当地时段的排放因子最后决定物理碳排。

在边界和统计周期一致的简化条件下，可以写成：

```text
单位任务设施电量 ≈ 单位任务 IT 电量 × PUE

单位任务物理碳排
≈ 单位任务 IT 电量 × PUE × 适用排放因子
```

因此，同一 PUE 下，只要单位任务 IT 电量或电网排放因子不同，单位任务碳排就会不同。绿电、绿证和市场化核算属于另一层口径，不能替代物理排放因子。

排放因子表示每使用一单位电量对应多少碳排放，常用单位为 `kgCO2e/kWh` 或 `tCO2e/MWh`。例如同样使用 `100 kWh`，排放因子分别为 `0.3` 和 `0.7 kgCO2e/kWh` 时，物理碳排分别约为 `30` 和 `70 kgCO2e`。平均排放因子用于描述整体平均供电结果，边际排放因子用于估计新增或迁移负荷的额外碳影响。

## 三类判断边界

| 类型 | 本轮内容 |
| --- | --- |
| 可以直接计算 | 在设施总用电、IT 用电和统计边界一致时，可计算 PUE。 |
| 必须实测或建模 | 不同负荷、天气、冷却方式和控制策略下的运行 PUE。 |
| 必须由业务、合同、制度或标准定义 | PUE 的测量边界、统计周期、测点和用于评价或披露的口径。 |
